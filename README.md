mlb
===================

CLI utility that prints baseball scores in your terminal

Requirements
===================

Python 3.6>= for f-Strings

Example
===================

```
-> ./mlb.py -t TEX

  ____________________________
 | Reds        |   0|   0|   0|   4:05 ET
 |----------------------------|
 | Rangers     |   0|   0|   0|
  ‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾

```
